# 🍔 MunchQuest

MunchQuest is a full-stack restaurant review and discovery web application where users can explore, rate, and manage restaurant listings. Designed for both regular users and admins, the platform supports authentication, review management, menu management, and more.

Live Site :(https://munchquest.vercel.app)

---

## 🧩 Features

- 🔍 Browse restaurants with ratings and images
- 📝 Leave reviews with star ratings
- 🧑‍💻 Admin and owner controls for adding, editing, or deleting restaurants
- 🧾 Menu editing support for restaurant owners and admins, enabling a fully customizable and dynamic menu experience per restaurant. 
- 🔐 Authentication via Firebase
- 🧠 Real-time UI feedback with loading spinners and toast notifications
- 📱 Responsive UI optimized for mobile and desktop

---

## 🚀 Tech Stack

**Frontend**
- React
- TailwindCSS
- Axios
- Vite

**Backend**
- Express.js
- MongoDB (via Mongoose)
- Cloudinary (for image storage)
- Firebase Admin SDK (for role-based claims)
- Method Override (for RESTful method emulation)

**Deployment**
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

---

## 🧪 Future Improvements

* Pagination for reviews/restaurants
* Search & filters for restaurants
* More robust admin dashboard
